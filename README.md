# 🌿 Biodiversity Management System - U Minh Thuong National Park

## 📌 Introduction
This project develops a web-based system for managing biodiversity data in **U Minh Thuong National Park**.  
The system supports storing, searching, analyzing, and visualizing information about flora, fauna, ecosystems, and conservation projects.

---

## 🎯 Objectives
- Centralize biodiversity data management  
- Support advanced search and data analysis  
- Provide interactive maps and data visualization  
- Enhance conservation awareness and community participation  

---

## 🚀 Key Features

### 1. Biodiversity Data Management
- Manage data of flora, fauna, ecosystems, and conservation projects  
- Easy data storage, update, and organization  

### 2. Advanced Search & Lookup
- Search species by:
  - Name  
  - Conservation status  
  - Geographic distribution  
- Fast and accurate results using Elasticsearch  

### 3. Interactive Ecological Map
- Display species distribution on maps  
- Explore biodiversity visually using Leaflet & ArcGIS  

### 4. Reports & Statistics
- Generate reports on biodiversity changes over time  
- Analyze ecosystem trends  

### 5. Data Analysis
- Perform large-scale data analysis  
- Support scientific research and conservation decisions  

### 6. Data Visualization
- Visualize data via:
  - Charts  
  - Graphs  
  - Maps  
- Powered by Kibana  

### 7. Role-Based Access Control
- User roles:
  - Admin  
  - Researcher  
  - Conservationist  
  - Visitor  
- Secure and appropriate data access  

### 8. Community Contributions
- Users can submit:
  - Species observations  
  - Conservation data  
- Helps enrich and update the database  

### 9. Conservation & Education
- Provide information about:
  - Conservation projects  
  - Environmental protection activities  
- Raise public awareness
